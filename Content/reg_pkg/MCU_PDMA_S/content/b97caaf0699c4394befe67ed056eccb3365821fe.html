Description

Secure Status Register

This register provides information about the overall status of the Secure channels.

Field Descriptions

RAZWI_1

Read-As-Zero, Writes Ignored.

STAT_CTXERR

Secure Virtual Channel Context Memory Access Error Status. Set to '1' whenever accessing the Secure virtual channel context memory results in an error. Cleared by writing '1' to this bit.

STAT_ALLCHPAUSED

All Secure Channel Paused Status. Set to '1' whenever all Secure DMA channel is in paused or inactive state and the SEC_CTRL.ALLCHPAUSE request is active. Not set when channels reach the paused state for other reasons. Cleared by writing '1' to this bit.

STAT_ALLCHSTOPPED

All Secure Channel Stopped Status. Set to '1' whenever all Secure DMA channels are in stopped or inactive state and the SEC_CTRL.ALLCHSTOP request is active. Not set when channels reach the stopped for other reasons. Cleared by writing '1' to this bit.

STAT_ALLCHIDLE

All Secure Channel Idle Status. Set to '1' whenever all Secure DMA channel is in idle state after at least one channel was running. Cleared by writing '1' to this bit.

RAZWI_0

Read-As-Zero, Writes Ignored.

INTR_CTXERR

Secure Virtual Channel Context Memory Access Error Interrupt Status. Set to '1' when SECCTRL.STAT_CTXERR is asserted and SECCTRL.INTREN_CTXERR = 1. Cleared automatically when STAT_CTXERR is cleared.

INTR_ALLCHPAUSED

All Secure Channel Paused Interrupt Status. Set to '1' when SECCTRL.STAT_ALLCHPAUSED is asserted and SECCTRL.INTREN_ALLCHPAUSED = 1. Cleared automatically when STAT_ALLCHPAUSED is cleared.

INTR_ALLCHSTOPPED

All Secure Channel Stopped Interrupt Status. Set to '1' when SECCTRL.STAT_ALLCHSTOPPED is asserted and SECCTRL.INTREN_ALLCHSTOPPED = 1. Cleared automatically when STAT_ALLCHSTOPPED is cleared.

INTR_ALLCHIDLE

All Secure Channel Idle Interrupt Status. Set to '1' when SECCTRL.STAT_ALLCHIDLE is asserted and SECCTRL.INTREN_ALLCHIDLE = 1. Cleared automatically when STAT_ALLCHIDLE is cleared.

INTR_ANYCHINTR

Combined Secure Channel Interrupt Flag. Set to '1' when any Secure channel has an interrupt request in CHINTRSTATUS0 or CHINTRSTATUS1 and SECCTRL.INTREN_ANYCHINTR = 1. Cleared automatically when the source of the channel interrupt is cleared.
