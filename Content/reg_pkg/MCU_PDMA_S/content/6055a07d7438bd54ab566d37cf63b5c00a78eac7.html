Description

Channel Status Register

This register shows the internal status of the DMA command and also reports interrupts about internal events.

Field Descriptions

RAZWI_3

Read-As-Zero, Writes Ignored.

STAT_TRIGOUTACKWAIT

Channel is waiting for output Trigger Acknowledgement Status. This bit is set to '1' when DMA starts waiting for output trigger acknowledgement. Automatically cleared when the output trigger acknowledgement is received either from from hardware or software source.

STAT_DESTRIGINWAIT

Channel is waiting for Destination Trigger Status. This bit is set to '1' when DMA starts waiting for destination input trigger request. Automatically cleared when the destination trigger request is received either from hardware or software source.

STAT_SRCTRIGINWAIT

Channel is waiting for Source Trigger Status. This bit is set to '1' when DMA starts waiting for source input trigger request. Automatically cleared when the source trigger request is received either from hardware or software source.

RAZWI_2

Read-As-Zero, Writes Ignored.

STAT_RESUMEWAIT

Waiting for resume from software Flag. This flag indicates that the DMA channel successfully paused the operation of the command and needs SW acknowledgment to resume the operation. Is set to '1' if STAT_PAUSED is asserted and the PAUSECMD bit set in the command register or when the STAT_DONE is asserted and the DONEPAUSEEN bit is set. Cleared when the RESUME bit is set in the command register.

STAT_PAUSED

Paused Status Flag. This flag is set to '1' if the DMA channel successfully paused the operation of the command. The pause request can come from many internal or external sources. When the request to pause is not asserted anymore the bit will be cleared automatically and the command operation can continue.

STAT_STOPPED

Stopped Status Flag. This flag is set to '1' if the DMA channel successfully reached the stopped state. The stop request can come from many internal or external sources. Write '1' to this bit to clear it. Automatically cleared when the ENABLECMD is set.

STAT_DISABLED

Disabled Status Flag. This flag is set to '1' if the DMA channel is successfully disabled using the DISABLECMD command. Write '1' to this bit to clear it. Automatically cleared when the ENABLECMD is set.

STAT_ERR

Error Status Flag. This flag is set to '1' if the DMA encounters an error during its operation. The details about the error event can be found in the ERRINFO register. When cleared, it also clears the ERRINFO register. Automatically cleared when the ENABLECMD is set.

STAT_DONE

Done Status Flag. This flag is set to '1' when the DMA command reaches the state defined by the DONETYPE settings. When DONEPAUSEEN is set the DMA command operation is paused when this flag is asserted. Writing '1' to this bit to clear it. Automatically cleared when the ENABLECMD is set.

RAZWI_1

Read-As-Zero, Writes Ignored.

INTR_TRIGOUTACKWAIT

Channel is waiting for output Trigger Acknowledgement Interrupt Flag. This interrupt is set to '1' if the INTREN_TRIGOUTACKWAIT is set and the STAT_TRIGOUTACKWAIT status flag is asserted. Automatically cleared when STAT_TRIGOUTACKWAIT is cleared.

INTR_DESTRIGINWAIT

Channel is waiting for Destination Trigger Interrupt Flag. This interrupt is set to '1' if the INTREN_DESTRIGINWAIT is set and the STAT_DESTRIGINWAIT status flag is asserted. Automatically cleared when STAT_DESTRIGINWAIT is cleared.

INTR_SRCTRIGINWAIT

Channel is waiting for Source Trigger Interrupt Flag. This interrupt is set to '1' if the INTREN_SRCTRIGINWAIT is set and the STAT_SRCTRIGINWAIT status flag is asserted. Automatically cleared when STAT_SRCTRIGINWAIT is cleared.

RAZWI_0

Read-As-Zero, Writes Ignored.

INTR_STOPPED

Stopped Interrupt Flag. This interrupt is set to '1' if the INTREN_STOPPED is set and the STAT_STOPPED flag gets raised. Automatically cleared when STAT_STOPPED is cleared.

INTR_DISABLED

Disabled Interrupt Flag. This interrupt is set to '1' if the INTREN_DISABLED is set and the STAT_DISABLED flag gets raised. Automatically cleared when STAT_DISABLED is cleared.

INTR_ERR

Error Interrupt Flag. This interrupt is set to '1' if the INTREN_ERR is set and the STAT_ERR status flag gets gets raised. Automatically cleared when STAT_ERR is cleared.

INTR_DONE

Done Interrupt Flag. This interrupt is set to '1' if the INTREN_DONE is set and the STAT_DONE status flag gets raised. Automatically cleared when STAT_DONE is cleared.
