Description

Non-Secure Unit Signal Pointer

This register can set a pointer to an internal register that shows the state of the Non-secure interfaces attached to the DMAC.

Contents

BitsIdentifierAccessResetDecodedName
[31:4] RAZWI_0 rw 0x0 - H

Field Descriptions

RAZWI_0

Read-As-Zero, Writes Ignored.

NSECSIGNALPTR

Non-secure DMA Unit signal pointer used to select which inputs or outputs to view using SIGNALVAL register. Pointer values are: - 0 : Source Trigger Input Requests [31:0]. Limited by NUM_CHANNELS, all unimplemented bits are reserved. - 1 : Destination Trigger Input Requests [31:0]. Limited by NUM_CHANNELS, all unimplemented bits are reserved. - 8 : Trigger output Acknowledges [31:0] - 10: GPO output value [31:0], only present when HAS_GPOSEL is set - 11: GPO output value [63:32], only present when HAS_GPOSEL is set - Others: Reserved
