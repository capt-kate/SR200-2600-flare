Description

Channel Error Information Register

This register provides information about internal errors encountered during command exectuion by the DMA channel.

Field Descriptions

ERRINFO

Error information Register. Additional information for the error that is encountered by the DMA. The meaning of the bits is IMPLEMENTATION DEFINED.

RAZWI_1

Read-As-Zero, Writes Ignored.

CTXERR

Virtual Channel Context Memory Access Error Flag. Set to '1' when accessing the context memory area of this channel results in an error.

RAZWI_0

Read-As-Zero, Writes Ignored.

CFGERR

Configuration Error Flag. Set when the DMA command is enabled or a linked command is read but it is configured in a mode that is not supported by the implementation.

BUSERR

Bus Error Flag. Set when the DMA encounters a bus error during data or command read transfers.
