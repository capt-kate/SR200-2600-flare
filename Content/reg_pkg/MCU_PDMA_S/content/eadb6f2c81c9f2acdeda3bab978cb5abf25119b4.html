Description

Channel Destination Trigger In Configuration Register

This register provides configuration settings when using destination side trigger input for the current DMA command.

The content of this register can be updated during command linking by setting bit[20] in the command link header.

Field Descriptions

RAZWI_2

Read-As-Zero, Writes Ignored.

DESTRIGINBLKSIZE

Destination Trigger Input Default Transfer Size. Defined transfer size per trigger + 1.

RAZWI_1

Read-As-Zero, Writes Ignored.

DESTRIGINMODE

Destination Trigger Input Mode: - 00: Command - 01: Reserved - 10: DMA driven Flow control. Only allowed when HAS_TRIGIN is enabled. - 11: Peripheral driven Flow control. Only allowed when HAS_TRIGIN is enabled. Note: This field is ignored for Internal triggers as they only support Command triggers

DESTRIGINTYPE

Destination Trigger Input Type: - 00: Software only Trigger Request. DESTRIGINSEL is ignored. - 01: Reserved - 10: Hardware Trigger Request. Only allowed when HAS_TRIGIN is enabled. DESTRIGINSEL selects between external trigger inputs if HAS_TRIGSEL is enabled. - 11: Internal Trigger Request. Only allowed when HAS_TRIGSEL is enabled and the DMAC has multiple channels, otherwise treated as HW Trigger Request. DESTRIGINSEL selects between DMA channels.

RAZWI_0

Read-As-Zero, Writes Ignored.
