Description

Secure Unit Signal Value Register

This register shows the current value of the Secure interfaces selected by the pointer in SEC_SIGNALPTR.

Contents

Field Descriptions

SECSIGNALVAL

Secure DMA Unit signal status. Can be used for reading signal values of triggers and GPOs for debug. Values shown here are dependent on SIGNALPTR. Note that inputs are masked by security mapping before being presented here. This means that only signal values mapped to Secure world are visible as non-zero values. Writing to this register is ignored. Note that value is also limited by NUM_CHANNELS. All unimplemented bits are RAZ/WI.
