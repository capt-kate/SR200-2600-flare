INTR_FIFO_TIMESTAMP_CTRL

32'h00000020

Range

Field

Reset

Description

[0:0]

en

0x0

Enables the timestamp counter.

[1:1]

clear

0x0

Clears the timestamp counter value and flushes the FIFOs.

[13:2]

rate

0x0

The timestamp logic runs on clk_cntr and this setting controls the rate at which the timestamp counter increases its count. When set to 0, it increases by 1 every 1 clock cycle. When set to 1, it increases by 1 every 2 clock cycles. When set to N-1, it increases by 1 every N clock cycles. When set to N, it increases by 1 every N+1 clock cycles.
