SUBCLOCK

Absolute Address:
Base Offset:

Description

Defines how and when returned data is to be sampled from the interface.

Field Descriptions

RESERVEDBITS

Reserved

RTD

RTD, or the round trip delay time. This is the number of clocks taken round trip for data to go through the PHY, interact with the external flash, and then be returned to the host controller. The default value is parameterized by DEF IO DELAY, and defaults to 1.

RESERVEDFIELDS

Reserved

High_speed_sub_clock_offset

High speed sub–clock offset. This determines which of the eight sub DEV CLK cycles is used to sample the incoming bit when the o_xspi_ck is running at the same rate as DEV CLK. An offset of zero returns the newest of the sub-clock bits. Defaults to 3.

Sub_clock_Offset

Sub_clock Offset. This is the offset used when the o xspi ck is running slower than DEV CLK. As with the high speed offset,seven selects the first value read, whereas zero selects the most recent. Defaults to 6.
