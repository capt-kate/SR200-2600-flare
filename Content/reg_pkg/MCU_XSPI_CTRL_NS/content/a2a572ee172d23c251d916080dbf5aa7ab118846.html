TSTSTATUS

Absolute Address:
Base Offset:
0x54
Size:
0x4

Description

This read-only 32b register returns the status of the latest PHY test.

Contents

Bits Identifier Access Reset Decoded Name
[31:0] Phy_test_status r - - H

Field Descriptions

Phy_test_status

Depending on the test, the result might be:

[31:24] == 8'h2 Synchronous BIST test result [23:18] (Reserved) [ 17] Set once the test has completed [ 16] Set following any error [15:14] (Reserved) [13: 9] Indicates either the current clock speed, if the test is ongoing, else the clock speed of any failure once the test has stopped. [ 8] Set to indicate a fault in the data strobe pin [ 7: 0] Set to indicate a fault found in the respective data pin [31:24] == 8'h3 Asynchronous BIST test result Bits [23:0] are defind as with the synchronous BIST results

All other tests will return a 32'h0 result.
