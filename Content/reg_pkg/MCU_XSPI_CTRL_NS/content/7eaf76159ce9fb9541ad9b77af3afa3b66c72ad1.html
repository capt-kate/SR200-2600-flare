Description

Defines timing related configuration, to include the speed of the SPI clock.

Field Descriptions

Disable_clock

Clock disable bit. If set, the clock will remain idle (zero) between device interactions. This is both the default value and the recommended setting. If clear, the IP will attempt to keep the clock running between commands, but may need to shut the clock down when starting or closing commands to avoid violating xSPI timing requirements.

RESERVED_FIELDS

Reserved

Rate_request

These bits are are a reflection of the rate request, indicating the current clock rate

Reserved

Reserved

Data_strobe

DS If set, this bit will turn off the normal mode of expecting a data strobe to be returned with any data in 4D or 8D modes.This bit has no effect in either 1S/1D or 2S/2D modes.

PH

Ph Reserved for controlling CPHA. Set to zero.

Pl

Pl Reserved for controlling CPOL. Set to zero.

Clock_divider_setting

Clock divider setting. Set to zero to run the device clock at the same speed as the incoming DEV CLK. Higer numbers will apply an integer clock division algorithm to divide the DEV CLK by twice the clock divider setting.
