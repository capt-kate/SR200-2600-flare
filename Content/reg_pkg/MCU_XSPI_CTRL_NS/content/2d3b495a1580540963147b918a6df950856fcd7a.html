Field Descriptions

Reserved

Reserved

Calibration_error

External resistor not present. This bit indicates a PHY calibration error condition.

Calibration_Done

Calibration done. On startup, the PHY will initiate a calibration cycle. During this time, any requests of the memory mapped channel will stall. Once complete, this bit will be set and requests will proceed to the device.

ReservedBits

Reserved

Calibration_Code

Calibration code. Once the PHY has completed its calibration cycle, this field will indicate the results of that calibration cycle. For more details on this field, see the ACS PHY user guide.
