Description

Secure Configuration Channel Security Mapping Register 0

This register is used to define the security attribute of each channel from Channel 0 to Channel 31.

Contents

BitsIdentifierAccessResetDecodedName
[31:0] SCFGCHSEC0 rw 0x0 - H

Field Descriptions

SCFGCHSEC0

Secure Configuration Channel Security Mapping for Channel 0 to 31. When set to '1', CH is Non-secure world, else Secure world. The NUM_CHANNELS parameter limits this field, unused bits are RAZWI. Value for bit can only be changed if the selected channel is not enabled, so reading back the register content is needed to check the success of the change.
