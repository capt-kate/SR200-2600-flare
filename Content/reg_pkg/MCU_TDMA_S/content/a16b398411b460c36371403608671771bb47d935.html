SEC_SIGNALPTR

Absolute Address:
Base Offset:
0x1f8
Size:
0x4

Description

Secure Unit Signal Pointer

This register can set a pointer to an internal register that shows the state of the Secure interfaces attached to the DMAC.

Contents

Bits Identifier Access Reset Decoded Name
[31:4] RAZWI_0 rw 0x0 - H
[3:0] SECSIGNALPTR rw 0x0 - H

Field Descriptions

RAZWI_0

Read-As-Zero, Writes Ignored.

SECSIGNALPTR

Secure DMA Unit signal pointer used to select which inputs to view using SIGNALVAL register. Pointer values, x, are: - 0 to 7: Trigger Input Requests [31+32x : 32x] - 8 to 9: Trigger output Acknowledges [31+32(x-8) : 32(x-8)] - 10 to 11: GPO output value [31+32(x-10) : 32(x-10)], only present when HAS_GPOSEL is set. - Others: Reserved
