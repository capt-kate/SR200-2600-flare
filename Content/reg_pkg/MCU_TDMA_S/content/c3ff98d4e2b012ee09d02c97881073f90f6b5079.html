CH_TMPLTCFG

Absolute Address:
Base Offset:
0x1040
Size:
0x4

Description

Channel Template Configuration Register

This register provides configuration settings when using template pattern based copy operations.

The content of this register can be updated during command linking by setting bit[16] in the command link header.

Contents

Bits Identifier Access Reset Decoded Name
[31:21] RAZWI_2 rw 0x0 - H
[20:16] DESTMPLTSIZE rw 0x0 - H
[15:13] RAZWI_1 rw 0x0 - H
[12:8] SRCTMPLTSIZE rw 0x0 - H
[7:0] RAZWI_0 rw 0x0 - H

Field Descriptions

RAZWI_2

Read-As-Zero, Writes Ignored.

DESTMPLTSIZE

Destination Template Size in number of transfers plus one. - 0: Destination template is disabled. - 1 to 31: DESTMPLT[DESTMPLTSIZE:0] is used as the destination template. Not present when HAS_TMPLT is 0.

RAZWI_1

Read-As-Zero, Writes Ignored.

SRCTMPLTSIZE

Source Template Size in number of transfers plus one. - 0: Source template is disabled. - 1 to 31: Bits SRCTMPLT[SRCTMPLTSIZE:0] is used as the source template. Not present when HAS_TMPLT is 0.

RAZWI_0

Read-As-Zero, Writes Ignored.
