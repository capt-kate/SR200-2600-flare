Description

Channel Implementation Identification Register

This register provides information about the revision of the product implementing the DMA channel.

Contents

Bits Identifier Access Reset Decoded Name
[31:20] PRODUCTID r 0x3a0 - H
[19:16] VARIANT r 0x0 - H
[15:12] REVISION r 0x0 - H
[11:0] IMPLEMENTER r 0x43b - H

Field Descriptions

PRODUCTID

Indicates the product ID

VARIANT

Indicates the major revision, or variant, of the product rxpy identifier

REVISION

Indicates the minor revision of the product rxpy identifier

IMPLEMENTER

Contains the JEP106 code of the company that implemented the IP: - [11:8]: JEP106 continuation code of implementer. - [7]: Always 0. - [6:0]: JEP106 identity code of implementer. For Arm this field reads as 0x43B.
