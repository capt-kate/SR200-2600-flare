Description

Channel Link Address Register, High Bits [63:32]

This register sets the upper address bits of the next element in a command link.

The content of this register can be updated during command linking by setting bit[31] in the command link header.

Contents

BitsIdentifierAccessResetDecodedName
[31:0] LINKADDRHI rw 0x0 - H

Field Descriptions

LINKADDRHI

Link Address Pointer [63:32]. Allows 64-bit addressing but the system might need less address bits. Limited by ADDR_WIDTH and the not implemented bits remain reserved.
