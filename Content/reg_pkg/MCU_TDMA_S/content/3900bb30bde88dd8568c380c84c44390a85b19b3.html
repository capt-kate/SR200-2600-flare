Absolute Address:
Base Offset:

Size:
0x4

Description

Channel Source Template Pattern Register

This register sets the template pattern used for reading the source memory area.

The content of this register can be updated during command linking by setting bit[17] in the command link header.

Contents

BitsIdentifierAccessResetDecodedName
[31:1] SRCTMPLT rw 0x0 - H
[0] SRCTMPLTLSB r 0x1 - H

Field Descriptions

SRCTMPLT

Source Packing Template. Bit[0] is read only and always set to 1 as template patterns can only start from the base address of the transfer. Not present when HAS_TMPLT is 0.

SRCTMPLTLSB

Source Packing Template Least Significant Bit. This bit of the template is read only and always set to 1 as template patterns can only start from the base address of the transfer. Not present when HAS_TMPLT is 0.
