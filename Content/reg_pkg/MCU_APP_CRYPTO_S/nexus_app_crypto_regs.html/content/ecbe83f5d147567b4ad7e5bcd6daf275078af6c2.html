When this bit is set, the AXI bus access range or module or register is not valid.

axi_sl_bus_err - AXI Slave Bus Error

When this bit is set, the AXI slave detects AXI bus errors such as bus width, transaction ID not matched, and bus protection error.

When this bit is set, the AXI slave Input FIFO detects some parity error.
