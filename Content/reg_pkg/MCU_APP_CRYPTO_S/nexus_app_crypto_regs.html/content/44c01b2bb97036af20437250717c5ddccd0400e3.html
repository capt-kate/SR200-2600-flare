Description

ZMP Operands

• M (APB Byte Address Offset: 100h)- Moduli for all ZMP operations.- Its value must be a prime (or odd) number.- M operand shall be larger than all other operands.

• X (APB Byte Address Offset: 104h)- Multiplicand for Montgomery Multiplication and Modular

Multiplication.- Base of exponentiation for Modular Exponentiation and Modular Inverse.- Addend for Modular Addition- Minuend for Modular Subtraction

• Y (APB Byte Address Offset: 108h)- Multiplicand for Montgomery Multiplication and Modular

Multiplication..- Summand for Modular Addition- Subtrahend for Modular Subtraction

• Z (APB Byte Address Offset: 10Ch)- Montgomery domain conversion parameter (R=2^2n Mod

M) for all ZMP operations, except Montgomery Multiplication and Pre-Computation.- Computation Result for all ZMP operations

Notes:

1. Operand should be programmed from LSB to MSB.

2. If a ZMP operation is performed right after the Pre-Computation and that operation requires the R, it is ok to not to unload/load the R after Pre-Computation because the Z and R are sharing the same memory space.
