Field Descriptions

out_dis_xmit_err_code - Output DMA DATA WRITE ERROR CHECK DISABLE

When this bit is asserted, the Output Link List Error check is disabled. The error interrupt is still set, but it does not affect the DMA operation.

out_dis_ll_err_code - Output DMA LINK LIST ERROR CHECK DISABLE

When this bit is asserted, the Output Link List Error check is disabled. The error interrupt is still set, but it does not affect the DMA operation.

out_dis_err_code - Output DMA DISABLE FIFO PARITY CHECK SELECT

When this bit is asserted, the Output FIFO Parity Error check is disabled. The error interrupt is still set, but it does not affect the DMA operation.

out_cache_mode - Output DMA Cache Mode Select

This field provides the CACHE mode for the DMA write transactions.

out_wid_ext - Output AXI Write ID

out_req_algn64 - Output DMA Request Data Aligned 64 Byte

out_req_pause - Output DMA Request Pause

out_bus_sz - Output DMA FIFO Bus Size

The default of this bit is product dependent.

This register provides the maximum bus size of the AXI output data bus. The value is indicated as follows:

0 - 32 bit bus

1 - 64 bit bus

out_burst_type - Output DMA Burst Type

out_burst_len - Output DMA Burst Length

Burst Length provides the number of beats of data transfer on the bus. The value can be from 1 to 16.

The total data transfer for each burst is calculated as follows:

Number of bytes = Burst Length x Bus Size (in terms of bytes)

out_wid - Output AXI Write ID Extension

out_prot_mode - Output DMA Protect Mode Select

This provides the PROT mode field for the DMA write transactions.

out_dma_ena - Output DMA Enable

When this bit is set, the Output DMA is enabled.
