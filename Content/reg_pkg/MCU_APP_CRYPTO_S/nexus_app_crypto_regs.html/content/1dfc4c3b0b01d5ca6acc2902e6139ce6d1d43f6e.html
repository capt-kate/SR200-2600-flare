Field Descriptions

reset - Soft Reset

Set this bit to soft reset Scratch Pad

zerostart - Memory Zerorize Start

Set this bit to zerorize the Scratch Pad internal SRAM. This bit is cleared automatically after it is set. The zerorize feature shall only be used when Scratch Pad is in CPU access mode.
