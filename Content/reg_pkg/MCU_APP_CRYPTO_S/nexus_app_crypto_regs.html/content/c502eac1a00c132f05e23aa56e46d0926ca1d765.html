Field Descriptions

in_dis_rec_err_code - Input DMA DATA READ ERROR CHECK DISABLE

When this bit is asserted, the Input data Transfer Error check is disabled. The error interrupt is still set, but it does not affect the DMA operation.

in_dis_ll_err_code - Input DMA LINK LIST ERROR CHECK DISABLE

When this bit is asserted, the Input Link List Error check is disabled. The error interrupt is still set, but it does not affect the DMA operation.

in_dis_err_code - Input DMA DISABLE FIFO PARITY CHECK SELECT

When this bit is asserted, the Input FIFO parity check is disabled. The Parity interrupt is still set, but it does not affect the DMA operation.

in_cache_mode - Input DMA Cache Mode Select

This field provides the CACHE mode for the DMA read transactions.

in_rid_ext - Input AXI Read ID Extension

This field provides the upper four bit extension of the RID for the read Link List RID. The subsequent RID+1 thru RID+4 will be used for the four reserved RID's for read data transfers.

The IN_RID should be different from OUT_WID and OUT_WID+1.

in_order_dis - Read Transaction Out of Order Disable

When this bit is set, the read transaction out of order feature is disabled, and the RID will always be the same for each read transaction.

in_req_algn64 - Input DMA Request Data Aligned 64 Byte

in_req_pause - Input DMA Request Pause

in_bus_sz - Input DMA FIFO Bus Size

The default of this bit is product dependent.

This register provides the maximum bus size of the AXI input data bus. The value is indicated as the following:

0 - 32 bit bus

1 - 64 bit bus

in_burst_type - Input DMA Burst Type

This field provides the Burst Type of the Read Data Transfer.

00 - Fixed

01 - Inc

10 - Wrap

11 - Reserved

in_burst_len - Input DMA Burst Length

Burst Len provides the #no beat of data transfer on the bus. The value can be from 1 thru 16.

The total data transfer for each burst is calculated as the following:

Number of bytes = #Burst Size X #Bus_Size (in term of bytes)

in_rid - Input AXI Read ID

This field provides the RID for the read Link List RID. The constraints for assignment of RID are the following:

Case1: Out-of-Order Read transfer is selected, and maximum burst size is less than 64 bytes, then

• RID+1 thru RID+4 will be used for the four reserved RID for read data transfers.

• RID+4 can not be larger than 15.

Case2: Out-of-Order Read transfer is selected, and maximum burst size is larger than 64 bytes, then

• RID+1 thru RID+2 will be used for the four reserved RID for read data transfers.

• RID+2 can not be larger than 15.

Case3: Out-of-Order Read transfer is not selected,

• RID+1 will be used for the reserved RID for read data transfers.

• RID+1 can not be larger than 15.

The assignment of RID range should be different from WID and WID+1.

in_prot_mode - Input DMA Protect Mode Select

This provides the PROT mode field for the DMA read transactions.

in_dma_ena - Input DMA Enable

When this bit is set, the Input DMA is enabled. This bit must be set only after all other registers are setup including RID. The sequence is:

1. Setup all registers, block count, etc., including IN_RID.

2. Set IN_DMA_ENA to high to start the transfer.
