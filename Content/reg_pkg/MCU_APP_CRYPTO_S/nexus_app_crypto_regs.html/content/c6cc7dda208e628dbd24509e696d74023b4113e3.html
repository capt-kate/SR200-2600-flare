Field Descriptions

txerr - Data Transfer Error Interrupt

This bit is set when there is an uncorrectable data error during memory access. This includes the firmware access through AMC_ACC_XXX_DATA registers and hardware access during the operation. Write one to clear this bit.

operr - Invalid Operand Error Interrupt

This bit is set when engine detects invalid operand(s) value that can cause unexpected result. When the error occurs, the engine stops the operation. All given operands are lost. Write one to clear this bit.

done - Done Interrupt

This bit is set when engine has finished the given task. Write one to clear this bit.
