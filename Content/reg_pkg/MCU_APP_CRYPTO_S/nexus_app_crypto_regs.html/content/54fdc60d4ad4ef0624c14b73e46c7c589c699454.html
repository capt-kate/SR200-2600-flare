Field Descriptions

merr - Access Mode Error Interrupt

This bit is set when CPU accesses the Scratch Pad in non-CPU Access mode. Write one to clear the bit. • 1'h0 = No error • 1'b1 = Invalid access Notes: 1. Once this bit is set, only the firmware can clear it.

perr - Parity Error Interrupt

This bit is set when error occurs during CPU access. Write one to clear the bit. • 2'b00 = No error • 2'b01 = 1-bit error has occurred before • 2'b10 = 2-bit error has occurred before • 2'b11 = Both 1-bit and 2-biti has occurred before. Notes: 1. If 1-bit error is detected, the read data would be corrected automatically. If 2-bit error is detected, the read data remains as is (error is uncorrectable). 2. Once this bit is set, only firmware can clear it.
