SPAD_FCT_WPTR

Absolute Address:
Base Offset:
0x20
Size:
0x4

Description

Write pointer for Sequential and 8-Byte Reverse access modes

• The difference between read and write pointer shall not exceed Scratch Pad's size.

• Write pointer always points to the next available address space.

1. If firmware wants to modify the FIFO pointers, it is required to program both read and write pointers in Sequential access mode.

2. The write pointer shall be programmed before programming the read pointer

3. The access mode selection must be done before changing the pointer values.
