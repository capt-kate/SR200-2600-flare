Description

Non-Secure Channel Interrupt Status Register 0

This register shows the overall interrupt status of every Non-secure channel from Channel 0 to Channel 31.

Field Descriptions

CHINTRSTATUS0

Collated Non-Secure Channel Interrupt flags for channel 0 to channel 31. Limited by NUM_CHANNELS, all unimplemented bits are reserved.
