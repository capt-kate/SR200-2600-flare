Description

Channel Destination Address Register, High Bits [63:32]

This register defines the upper 32 bits of the write address in case more than 32-bit addressing is used in the system. When the register is read during the execution of the command it shows the actual address the DMA channel is going to write to.

The content of this register can be updated during command linking by setting bit[7] in the command link header.

Field Descriptions

DESADDRHI

Destination Address[63:32]. Allows 64-bit addressing but the system might need less address bits defined by ADDR_WIDTH. The not implemented bits remain reserved.
