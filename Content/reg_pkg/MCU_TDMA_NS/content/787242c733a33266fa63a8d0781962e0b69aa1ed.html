Description

DMA Build Configuration Register 0

This register is used to provide information about the configuration parameters of the DMAC.

Contents

BitsIdentifierAccessResetDecodedName

[2:0] FRAMETYPE r 0x0 - H

Field Descriptions

RAZWI_2

Read-As-Zero, Writes Ignored.

CHID_WIDTH

Channel ID Width. '0' means CHID is not present.

RAZWI_1

Read-As-Zero, Writes Ignored.

DATA_WIDTH

Data Width. - 000: 8-bit - 001: 16-bit - 010: 32-bit - 011: 64-bit - 100: 128-bit - 101: 256-bit - 110: 512-bit - 111: 1024-bit

ADDR_WIDTH

Address Width in bits = ADDR_WIDTH + 1

NUM_CHANNELS

Number of Channels + 1 supported by the DMAC

RAZWI_0

Read-As-Zero, Writes Ignored.

FRAMETYPE

Register Frame Type. - 000: Combined Frame, - 001: Security Configuration Frame, - 010: Secure Control Frame, or if TrustZone not implemented, Control Frame. - 011: Non-Secure Control Frame, - 100: DMA Channel Frame. Note that each frame replicates the Information fields.
