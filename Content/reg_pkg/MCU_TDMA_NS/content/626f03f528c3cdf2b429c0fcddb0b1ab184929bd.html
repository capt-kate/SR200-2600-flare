Description

Peripheral ID4 Register

This register returns byte[4] of the peripheral ID.

Contents

Bits Identifier Access Reset Decoded Name
[31:8] RAZWI_0 r 0x0 - H
[7:4] SIZE r 0x0 - H
[3:0] DES_2 r 0x4 - H

Field Descriptions

RAZWI_0

Read-As-Zero, Writes Ignored.

SIZE

4KB Count - the number of 4K pages used. - 0x00: 4K - 0x01: 8K - 0x02: 16K - 0x03: 32K

DES_2

JEP106 Continuation Code
