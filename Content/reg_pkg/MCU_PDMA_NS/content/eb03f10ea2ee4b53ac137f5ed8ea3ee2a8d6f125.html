Description

Channel Interrupt Enable Register

This register can enable the interrupt generation for internal events.

The content of this register can be updated during command linking by setting bit[2] in the command link header.

Field Descriptions

RAZWI_1

Read-As-Zero, Writes Ignored.

INTREN_TRIGOUTACKWAIT

Channel is waiting for output Trigger Acknowledgement Interrupt Enable. When set to '1', enables INTR_TRIGOUTACKWAIT to be set and raise an interrupt when STAT_TRIGOUTACKWAIT status flag is asserted. When set to '0', it prevents INTR_TRIGOUTACKWAIT to be asserted. Currently pending interrupts are not affected by clearing this bit.

INTREN_DESTRIGINWAIT

Channel is waiting for destination Trigger Interrupt Enable. When set to '1', enables INTR_DESTRIGINWAIT to be set and raise an interrupt when STAT_DESTRIGINWAIT status flag is asserted. When set to '0', it prevents INTR_DESTRIGINWAIT to be asserted. Currently pending interrupts are not affected by clearing this bit.

INTREN_SRCTRIGINWAIT

Channel is waiting for source Trigger Interrupt Enable. When set to '1', enables INTR_SRCTRIGINWAIT to be set and raise an interrupt when STAT_SRCTRIGINWAIT status flag is asserted. When set to '0', it prevents INTR_SRCTRIGINWAIT to be asserted. Currently pending interrupts are not affected by clearing this bit.

RAZWI_0

Read-As-Zero, Writes Ignored.

INTREN_STOPPED

Stopped Interrupt Enable. When set to '1', enables INTR_STOPPED to be set and raise an interrupt when STAT_STOPPED status flag is asserted. When set to '0', it prevents INTR_STOPPED to be asserted. Currently pending interrupts are not affected by clearing this bit.

INTREN_DISABLED

Disabled Interrupt Enable. When set to '1', enables INTR_DISABLED to be set and raise an interrupt when STAT_DISABLED status flag is asserted. When set to '0', it prevents INTR_DISABLED to be asserted. Currently pending interrupts are not affected by clearing this bit.

INTREN_ERR

Error Interrupt Enable. When set to '1', enables INTR_ERR to be set and raise an interrupt when the STAT_ERR status flag is asserted. When set to '0', it prevents INTR_ERR to be asserted. Currently pending interrupts are not affected by clearing this bit.

INTREN_DONE

Done Interrupt Enable. When set to '1', enables the INTR_DONE to be set and raise an interrupt when the STAT_DONE status flag is asserted. When set to '0', it prevents INTR_DONE to be asserted. Currently pending interrupts are not affected by clearing this bit.
