Description

DMA Build Configuration Register 2

This register is used to provide information about the configuration parameters of the DMAC.

Contents

BitsIdentifierAccessResetDecodedName

Field Descriptions

RAZWI_1

Read-As-Zero, Writes Ignored.

NUM_PCH

Number of Physical Channels. If HAS_VCH is '0' then this field is also '0'.

HAS_CHCTRL_MASK

Secure and Non-secure Control as triggers, GPO and Stream Interface Masking.

HAS_VCH

Has allocable DMA Channel Support.

HAS_RET

Has Retention Support

HAS_TZ

Has TrustZone Support

HAS_GPOSEL

Has Shared and hence Selectable GPO Support.

RAZWI_0

Read-As-Zero, Writes Ignored.
