Field Descriptions

outputdis - Output Disable

Set this bit to stop AES from outputting data on data bus. Some applications, such as DRBG BCC function, are only interested in the last ciphertext (CTX) of encryption result. Since the current design always saves the last CTX in the AES IV register, it is unnecessary to output the entire encryption result on data bus (save power and memory space).

Notes:

1. The input date stream must be multiple of 128-bits (no partial codeword allowed)

2. Resume operation is supported.

resume - Resume Operation

Set this bit to resume previous operation.

Notes:

1. User needs to load the saved configurations, KEY1 and IV back to corresponding registers.

2. Resume operation is not supported in Key Wrap and GCM mode.

3. Refer to AES_STR_SIZE register for data size limitation
