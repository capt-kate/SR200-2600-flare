Field Descriptions

ignperr - Ignore Memory Parity Error

frcperr - Force Memory Parity Error

Program this field to inject error into the parity bits during a read/pop operation.

• 2'b00 = No Error Injection

• 2'b01 = 1-bit error injection

• 2'b10 = 2-bit error injection

• 2'b11 = Reserved
