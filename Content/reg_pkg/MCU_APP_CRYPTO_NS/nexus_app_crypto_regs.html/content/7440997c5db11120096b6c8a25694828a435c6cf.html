Description

This register is used to configure Hash engine.

Contents

Bits Identifier Access Reset Decoded Name

Field Descriptions

rkek - RKEK Enable

This bit determines whether RKEK should be used as HMAC key. When this bit is set, RKEK is loaded into the HMAC key register. Read/ Write to the key register would be blocked until this bit is cleared. Notes: 1. HMAC clears the key register automatically when this bit is cleared 2. After every HMAC task that uses RKEK is done, this bit must be cleared. It can be set again if the next HMAC task still uses RKEK. 3. When RKEK is used, the max key length is limited to 32 bytes.

byte_swap - Byte Swap

0: Byte Swap disabled. The input data is in Big Endian format. The output digest is also in Big Endian format. 1: Byte Swap enabled. The input data is in Little Endian format. The ouput digest is also in Little Endian format.

output_data_mode - Output Data Mode

1: Hash-AES combine mode: output in-data-stream to output-datastream 0: Not HASH-AES combine mode: no output in-data-stream

hash_mode - Hash Mode

0: Simple hash mode 1: HMAC

alg_select - Hash Algorithm Select

Select hash algorithm be used from MD5, SHA-1, SHA-224, SHA-256, SHA-512, SHA-384, SHA-512/224, SHA-512/256 000: SHA-1 001: SHA-256 010: SHA-224 011: MD5 100: SHA-512 101: SHA-384 110: SHA-512/224 111: SHA-512/256
