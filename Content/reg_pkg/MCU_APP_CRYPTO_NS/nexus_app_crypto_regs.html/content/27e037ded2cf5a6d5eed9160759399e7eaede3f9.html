SPAD_ACC_DATA

Absolute Address:
Base Offset:
0x54
Size:
0x4

Description

The field is the read/write data in CPU access mode. It is only valid in CPU access mode. The standard procedure for reading/writing the Scratch Pad is the following.

1. Program SPAD_ACC_ADDR.

2. Read/Program SPAD_ACC_DATA.

The step 2 triggers the hardware to perform requested operation. The hardware is implemented with an auto-increment feature which automatically increases the SPAD_ACC_ADDR register by one after step 2. If the firmware wants to read/write a contiguous memory space, it just needs to provide the initial address to SPAD_ACC_ADDR register and then keeps reading/writing SPAD_ACC_DATA register.
