Field Descriptions

extctrlen - External Control Enable

Set this bit to enable external control. When external control is enabled, engine takes the following information form external source and ignores the configurations from the APB register.

• Start Signal

• Encryption/Decryption signal

• Key length

• Resume signal

• Payload/stream size

• Additional authenticated data size (GCM)

• IV size (GCM)

• Tag size (GCM)

• Key1

• Key2 (XTS)

• IV

dataswap - Data Swap

This field decides the Key1, Key2, IV, and input/output data byte ordering (endian) within a DWord.

• 0h= No swap (data feeded in as is)

• 1h= Byte swap

EX: Assuem a DWord data is 0x01234567

• 0h = 0x01234567.

• 1h = 0x67452301

Notes:

1. When External Control Enable bit is set, this field no longer has effect on the Key1, Key2, and IV.

2. This field must be programmed before programming the Key1, Key2, and IV registers in order to take effect.

select - AES, DES/TDES and RC4 share some of the registers and internal logic. The field allocates the resource to the selected engine.

• 0h= Reserved

• 1h="AES" engine

• 2h="DES/TDES" engine

• 3h="RC4" engine

Notes:

1. Both AES's IV and DES/TDES's IV are mapped to IV_DW0, IV_DW1, IV_DW2, and IV_DW3 registers. When writing to IV_DW registers, both AES's and DES/TDES's IV value are over-written. When read from IV_DW registers, the read value (AES's or DES/ TDES's IV) is selected by this field.
