Field Descriptions

axi_xmit_resp_max - AXI Maximum Outstanding Write Respond Queue

This field provides the maximum outstanding AXI Write Transaction Response Queue. When the maximum outstanding AXI Write Transaction Response Queue is not full, the new AXI write transaction can start after the current write data transfer completes w/o waiting for the response phase to complete.

axi_data_par_ena - AXI Data Parity Check Enable

When this is set, AXI Data Bus Parity check is enabled.

axi_wdata_par_even - AXI Write Data Even Parity

When this is set, AXI Write Data Bus Even Parity mode is selected.

axi_rdata_par_even - AXI Read Data Even Parity

When this is set, AXI Read Data Bus Even Parity mode is selected.

link_list_swap_dis - Disable Word/Byte Swap During Link List

When this is set, the IN_AXI_MS_BYTE_SWP and IN_AXI_MS_WRD_SWP are disable during Link List Read operation. This is useful when the Link Lists stored in the system memory as little endian compared to the big endian of the data format.

dma_loop_back_en - DMA Loop Back Enable

This bit is used for self test mode. When this bit is set, the loop back between Input DMA internal bus and the Output DMA internal bus is enabled.

out_axi_ms_wrd_swp - Master AXI Output Data Bus Word Swap Enable

When this bit is set, the upper and lower words of the 64 bit output data bus are swapped.

out_axi_ms_byte_swp - Master AXI Output Data Bus Byte Swap Enable

Within the 32-bit upper and lower words of the output data bus, the bytes are swapped. Bits [7:0] are swapped with bits [31:24], and bits [15:8] are swapped with bits [23:16].

in_axi_ms_wrd_swp - Master AXI Input Data Bus Word Swap Enable

When this bit is set, the upper and lower word sof the 64 bit input data bus are swapped.

in_axi_ms_byte_swp - Master AXI Input Data Bus Byte Swap Enable

Within the 32-bit upper and lower words of the output data bus, the bytes are swapped. Bits [7:0] are swapped with bits [31:24], and bits [15:8] are swapped with bits [23:16].
