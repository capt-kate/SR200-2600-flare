Hash Command

Absolute Address:
Base Offset:
0x8
Size:
0x4

Description

Set command to Hash Engine.

Contents

BitsIdentifierAccessResetDecodedName
[31:1] - - - -
[0] com_start rw 0x0 Hash Command Start H

Field Descriptions

com_start - Hash Command Start

Set this bit to set command start. Hash HW detect this bit's rising edge as command start pulse, once Hash engine starts, hash HW will auto clear this bit.
