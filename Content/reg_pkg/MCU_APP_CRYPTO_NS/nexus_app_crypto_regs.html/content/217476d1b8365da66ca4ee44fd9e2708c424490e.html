Field Descriptions

groupc_sel - Group C Accelerator Select

Group C accelerators are able to random access (read/write memory with particular address) the Scratch Pad. This field connects one of the accelerators in Group C to the Scratch Pad's random access port.

• 0h= AMC (Asymmetric Cipher Engine)

• 2h= RC4

• Else= Reserved

Notes:

1. When the Scratch Pad is not in the random access mode, this field is ignored.

2. Above selection is based on the default BCM hardware configuration. The actual selection may be different when the configuration is changed.

3. AMC is the engine that handles RSA and Elliptic Curve algorithm

groupb_sel - Group B Accelerator Select

Group B accelerators sequential access (using valid and ready signal to transfer data) the Scratch Pad and/or DMA. This field connects one of the accelerators in Group B to the Scratch Pad and/ or DMA.

• 0h= AES

• 1h= DES/TDES

• 2h= Bypass

• Else= Reserved

Notes:

1. The cross bar configuration decides how the data flows in and out of the Group B accelerator.

2. Above selection is based on the default BCM hardware configuration. The actual section may be different when configuration is changed.

groupa_sel - Group A Accelerator Select

Group A accelerators sequential access (using valid and ready signal to transfer data) the Scratch Pad and/or DMA. This field connects one of the accelerators in Group A to the Scratch Pad and/ or DMA.

• 0h= Hash

• 2h= RC4

• Else= Reserved

Notes:

1. The cross bar configuration decides how the data flows in and out of the Group A accelerator.

2. Above selection is based on the default BCM hardware configuration. The actual section may be different when configuration is changed.

wrcrossbar_sel - Write Cross Bar Select

This field connects the sequential access's output port of Group A and B to sequential access's input port of Scratch Pad and DMA.

• 0h= Straight connection. Data flows from Group A accelerator to Scratch Pad. Data flows from Group B Accelerator to DMA output buffer

• 1h= Cross connection. Data flows from Group B accelerator to Scratch Pad. Data flows from Group A accelerator to DMA output buffer

rdcrossbar_sel - Read Cross Bar Select

This field connects the sequential access's output port of Scratch Pad and DMA to sequential access's input port of Group A and B.

• 0h= Straight connection. Data flows from Scratch Pad to Group A accelerator. Data flows from DMA input buffer to Group B accelerator.

• 1h= Cross connection. Data flows from Scratch Pad to Group B accelerator. Data flows from DMA's input buffer to Group A accelerator.
