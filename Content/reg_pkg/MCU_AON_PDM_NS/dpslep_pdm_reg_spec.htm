dmic_retime _pdm

RW

RO

0

DMIC Retime module control. Each DM interface has three bits to control its functionality.

[0]: +ve FF Cycle Selection

0: Gets full cycle

1: Gets half cycle to avoid any Hold issues

[1]: -ve FF Cycle Selection

0: Gets full cycle

1: Gets half cycle to avoid any Hold issues

[2]: Bypass Retimer

0: Data is flopped using PAD clock loopback

1: Data is directly flopped using output Clock before PAD.

1 =Data driven by negative edge of DMIC clock on input DMIC interface is aligned with the data driven by following positive edge of DMIC clock.

2 =(support digital retiming logic)Data driven by positive edge of DMIC clock on input DMIC interface is aligned with the data driven by following negative edge of DMIC clock.
