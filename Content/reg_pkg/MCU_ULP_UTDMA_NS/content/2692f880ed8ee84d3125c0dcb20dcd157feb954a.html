Description

Non-Secure Unit Status Pointer Register

This register can set a pointer to an internal status register that can show the global state of the Non-secure channels.

Contents

Bits Identifier Access Reset Decoded Name
[31:4] RAZWI_0 rw 0x0 - H
[3:0] NSECSTATUSPTR rw 0x0 - H

Field Descriptions

RAZWI_0

Read-As-Zero, Writes Ignored.

NSECSTATUSPTR

Non-secure DMA Unit status pointer used to select which status value to view using STATUSVALUE register. Pointer values are: - 0: Channel Enabled Status for channel numbers [31 : 0]. - 1: Channel Enabled Status for channel numbers [63 : 32]. - 2: Channel Stopped Status for channel numbers [31 : 0]. - 3: Channel Stopped Status for channel numbers [63 : 32]. - 4: Channel Paused Status for channel numbers [31 : 0]. - 5: Channel Paused Status for channel numbers [63 : 32]. - 6: Channel Allocation Status for channel numbers [31 : 0]. Only implemented if HAS_VCH feature is supported. It shows if a virtual channel is granted allocation to a physical channel. - 7: Channel Allocation Status for channel numbers [63 : 32]. Only implemented if HAS_VCH feature is supported. - 8: Channel Allocation Stalled Status for channel numbers [31 : 0]. Only implemented if HAS_VCH feature is supported. It shows if a virtual channel is ready to execute the DMA job (it is requesting physical channel allocation), but it is not yet allocated. - 9: Channel Allocation Stalled Status for channel numbers [63 : 32]. Only implemented if HAS_VCH feature is supported. - Others: Reserved.
