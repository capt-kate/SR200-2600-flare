Description

The MODE register is used to enable or disable certain run-time features within the DWC TRNG NIST. This register can only be written when the core is not busy (see STAT.BUSY). the SEC_ALG field indicates the security algorithm for the DRBG, which determines the key size of the AES function. A SEC_ALG of 1 is only valid if the core is built with a 256-bit AES. If the AES module is an AES-128, the SEC_ALG never changes to one and a write of 1 to this field is ignored. This field must not change when the DRBG state is instantiated (that is, Create_State command has created an state for the DRBG). Any write to the SEC_ALG field while the DRBG state is instantiated is treated as a NOP.
