STAT

Absolute Address:
Base Offset:
0xc
Size:
0x4

Description

The STAT register allows the user to monitor the internal status of the DWC TRNG NIST.

NOTE: After the MODE.NONCE bit is set, the host needs to poll the NONCE_MODE in the STAT register until it is 1. Then, the host can load the nonce value into the core.

BUSY

Indicates the state of the core.

enum BUSY_enum_t
BUSY_EXEC 0x1

Currently executing a command

Idle

STARTUP_TEST_IN_PROG

Indicates whether the startup test is in progress.

enum STARTUP_TEST_IN_PROG_enum_t
STARTUP_TEST_IS_STUCK 0x1

Startup test is in progress.
STARTUP_TEST_NOT_STUCK 0x0

Startup test is finished.

STARTUP_TEST_STUCK

Indicates whether the startup test is stuck (Only valid when sticky startup test feature is enabled.

enum STARTUP_TEST_STUCK_enum_t
STARTUP_TEST_IS_STUCK 0x1

Startup test is highly likely stuck
STARTUP_TEST_NOT_STUCK 0x0

Startup test is not stuck (yet)

DRBG_STATE

Reflects how a DRBG state is instantiated.

enum DRBG_STATE_enum_t
DRBG_HOST 0x2

State is instantiated using the host-provided nonce
DRBG_NOT_INIT 0x0

State is not instantiated
DRBG_NS 0x1

State is instantiated using the built-in noise source
