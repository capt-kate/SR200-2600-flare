Description

The FEATURES register returns the state of various build-time parameter values. NOTE: DEPRECATED - for backward compatibility only. Use BUILD_CFGx register(s) instead. This register will be DELETED in the next release of the core.

Field Descriptions

AES_256

Indicates whether the instantiated AES is 128-bit or 256-bit.

PS_PRESENT

Determines whether the content of the NPA_DATAx registers is used as a Personalization String during the Create_State command.

DIAG_LEVEL_NS

Level of diagnostic circuitry for Noise Source output Registers.

DIAG_LEVEL_CLP800

Level of diagnostic circuitry for DWC_trng (Noise Source).

DIAG_LEVEL_ST_HLT

Level of diagnostic circuitry for the health test.

SECURE_RST_STATE

Indicates if the core resets to MISSION or TEST mode.
