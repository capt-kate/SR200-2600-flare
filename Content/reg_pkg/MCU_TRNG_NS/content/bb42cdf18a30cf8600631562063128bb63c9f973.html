ISTAT

Absolute Address:
Base Offset:
0x14
Size:
0x4

Description

The ISTAT register allows the user to monitor the interrupt contributions of the DWC TRNG NIST.

Note that these status signals are used for both polling mode and interrupt modes. Whether they assert the O_irq output depends on whether the IE.GLBL_EN and one or more of the other individual IE bits are set to 1. ZEROIZED and KAT_COMPLETED bits are only set after a command has explicitly requested for a RUN_KAT or ZEROIZE operation. During an alarm event or a start-up test, in which the core automatically performs a zeroize operation or runs a KAT, ZEROIZED and KAT_COMPLETED interrupts does not occur. ZEROIZED can also happen after a zeroize operation request through the input zeroize pin (I_ctl_zeroize). Zeroize operatoin through the input pin is not completed until the pin is de-asserted.

Field Descriptions

DONE

The DONE bit allows the user to poll the completion of all commands except RUN_KAT and ZEROIZE which have their own interrupt.

enum DONE_enum_t
DONE_R0 0x0

R0: No unacknowledged command completion
DONE_R1 0x1

R1: Unacknowledged command completion

ALARMS

The ALARMS bit allows the user to poll failures. When an alarm occurs, an automatic zeroize happens. Clearing this interrupt also clears the O_alarm pin.

enum ALARMS_enum_t
ALARMS_R0 0x0

R0: No unacknowledged ALARMS
ALARMS_R1 0x1

R1: Unacknowledged ALARMS

NOISE_RDY

When DWC TRNG NIST is generating a full-entropy seed in the self-seeding mode, SMODE.MISSION_MODE is 0 (TEST mode) and SMODE.NOISE_COLLECT is set to 1, the NOISE_RDY bit informs the user when 512 bits of noise have been generated. This interrupt never happens in the MISSION mode of operation.

enum NOISE_RDY_enum_t
NOISE_RDY_R0 0x0

R0: No unacknowledged noise generation completion
NOISE_RDY_R1 0x1

R1: Unacknowledged noise generation completion

KAT_COMPLETED

Indicates the completion of the RUN_KAT command.

enum KAT_COMPLETED_enum_t
KAT_COMPLETED_R0 0x0

R0: No unacknowledged KAT_COMPLETED
KAT_COMPLETED_R1 0x1

R1: Unacknowledged KAT_COMPLETED

ZEROIZED

Indicates the completion of the ZEROIZE operation.
