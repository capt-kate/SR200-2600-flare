Field Descriptions

auth_key_sel

Select an authentication key for this region by specifying the start byte address in the key table divided by 8.

auth_key_len

Select one of the 3 supported key lengths for authentication.

enum key_len_t
AES128 0x0

AES cipher with 128-bit key
AES192 0x1

AES cipher with 192-bit key
AES256 0x2

AES cipher with 256-bit key

auth_en

Whether authentication is enabled for this region. 0: disabled, 1: enabled.

tag_en

Set to 1 if an 8-byte authentication tag is stored next to every 16-byte data. This implies the address space for this region on the upstream slave interface is remapped to a 1.5x larger address space on the downstream master interace. The start address of this region remains the same on both interfaces. This only specifies the storage format regardless of whether authentication is enabled or not.

decrypt_nonce_sel

Select a decryption nonce for this region by specifying the start byte address in the nonce table divided by 12.

decrypt_key_sel

Select a decryption key for this region by specifying the start byte address in the key table divided by 8.

decrypt_key_len

Select one of the 3 supported key lengths for decryption.

enum key_len_t
AES128 0x0

AES cipher with 128-bit key
AES192 0x1

AES cipher with 192-bit key
AES256 0x2

AES cipher with 256-bit key

decrypt_en

Whether decryption is enabled for this region. 0: disabled, 1: enabled.
