Field Descriptions

axi_data_par_err - AXI Data Parity Error

When this bit is set, the DMA transfer has detected an AXI input read data parity error. The Input DMA Enable bit must be reset before one can clear this interrupt. This bit is reset by writing a one to it.

in_dma_pause_cmpl - Input DMA Pause Complete

When this bit is set, the DMA is paused. This bit is set only when all current AXI read transactions are completed, and the Input DMA transfer count will reflect the actual amount of receive data transferred. The Input DMA FIFO count will show the remaining data in the FIFO. The Input DMA Request bit must be reset before one can clear this interrupt.

in_par_err - Input FIFO Parity Error

When this bit is set, the DMA transfer has detected an Input DMA FIFO parity error. The Input DMA Enable bit must be reset before one can clear this interrupt. This bit is reset by writing one a to it.

in_ll_err - Link List AXI Bus Access Error

When this bit is set, the Input DMA Link List read operation detected a bus error. The Input DMA Enable bit must be reset before one can clear this interrupt. This bit is reset by writing a one to it.

in_dma_bus_err - AXI Read Cycle Access Error Code

When this bit is set, the DMA transfer has detected a bus error. The Input DMA Enable bit must be reset before the user can clear this interrupt. This bit is reset by writing a one to it.

in_dma_done - Input DMA Transfer Complete

When this bit is set, the current DMA transfer is completed and all data has been received into the DMA input FIFO. It might take a while for the data to be transferred to the accelerator engines. The Input DMA Enable bit must be reset before one can clear this interrupt. This bit is reset by writing a one to it
