Field Descriptions

txerr - Data Transfer Error Interrupt

This bit is set when there is an uncorrectable data error during the data transfer. When the error occurs, the engine continues the current operation based on corrupted data. As soon as the engine finishes the operation (by checking either busy bit or done interrupt bit), firmware can clear this bit by writing one to this bit or trigger the start of the next operation.

tagerr - GCM Tag Error Interrupt

This bit is set when GCM tag mismatch occurs in GCM decryption. Write one to clear this bit. Triggering the start of the next operation also clears this bit.

lenerr - Length Error Interrupt This bit is set when:

• ECB Mode

• AES_STR_SIZE register is not multiple of 16 bytes

• AES_STR_SIZE register is 0 byte.

• CBC Mode

• AES_STR_SIZE register is less than 16 bytes

• CTR Mode

• AES_STR_SIZE register is 0 byte.

• XTS Mode

• AES_STR_SIZE register is less than 16 bytes

• Key Wrap Mode

AES_STR_SIZE register is not multiple of 8 bytes.

• AES_STR_SIZE register is more than Scratch Pad can handle

• AES_STR_SIZE register is 0 byte.

• GCM Mode

• AES_IV_SIZE register is 0 byte

• AES_TAG_SIZE register is 0 byte

Write one to clear this bit. Triggering the start of the next operation also clears this bit.

done - Done Interrupt

This bit is set when AES has finished the given task. Write one to clear this bit. Triggering the start of the next operation also clears this bit.
