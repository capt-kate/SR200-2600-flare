Field Descriptions

axi_data_par_err - AXI Data Parity Error

When this bit is set, the DMA transfer has detected an AXI input read data parity error during an Output DMA Link List request cycle. The Output DMA Enable bit must be reset before one can clear this interrupt. This bit is reset by writing a one to it.

out_dma_pause_cmpl - Output DMA Pause Complete

When this bit is set, the Output DMA is paused.

The Output DMA Request bit must be reset before one can clear this interrupt. This bit can be reset by writing a one to it.

out_par_err - Input FIFO Parity Error

When this bit is set, the Output DMA detected a FIFO Parity error. The Output DMA Enable control must be reset before this bit can be cleared. This bit is reset by writing a one to it.

out_ll_err - Link List AXI Bus Access Error

When this bit is set, the Output DMA Link List read operation detected a bus error.

The Output DMA Enable control must be reset before this bit can be cleared. This bit is reset by writing a one to it.

out_dma_bus_err - AXI Read Cycle Access Error Code

When this bit is set, the Output DMA operation detected a bus error. The Output DMA Enable control must be reset before this bit can be cleared. This bit is reset by writing a one to it.

out_dma_done - Input DMA Transfer Complete

When this bit is set, the current DMA operation is completed. The Output DMA Enable control must be reset before this bit can be cleared. This bit is cleared by writing a one to it.
