Field Descriptions

pause_cmpl - Output DMA Pause Complete

When this bit is set, the Output DMA is paused.

ll_err_code - Write Linked List Error Code

This field provides the Write Link List Transfer Error on the bus.

00 - No Error

01 - No EX Error

10 - Slave Error

11 - Decode Error

par_err - Output FIFO Parity Error

When this bit set, the Output DMA FIFO has data parity error.

wr_cycle_err - Write Cycle Access Error Code

This field provides the Write Data Transfer Error on the bus.

00 - No Error

01 - No EX Error

10 - Slave Error

11 - Decode Error
