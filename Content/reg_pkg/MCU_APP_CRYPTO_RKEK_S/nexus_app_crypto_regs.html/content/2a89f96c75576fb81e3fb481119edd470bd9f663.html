DMA Input FIFO Remain Count

Absolute Address:
Base Offset:
0x3c
Size:
0x4

Description

This register indicates how much data remains in the DMA Input FIFO. This register and the Input DMA Counter can be used to calculate how much data has been sent to the encryption engine.
