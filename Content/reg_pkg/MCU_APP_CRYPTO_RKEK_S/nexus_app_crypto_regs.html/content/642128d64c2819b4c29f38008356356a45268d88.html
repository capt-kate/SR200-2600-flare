Absolute Address:
Base Offset:

Description

Initialization Vector (IV) Size (in Byte)

• Non-GCM Mode

• The field is ignored because AES always uses 16 bytes IV in non-GCM mode automatically.

• GCM mode

• The size must be larger than 0 byte.
