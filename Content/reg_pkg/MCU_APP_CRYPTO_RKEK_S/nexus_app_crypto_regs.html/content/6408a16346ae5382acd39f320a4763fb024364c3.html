DES_STR_SIZE

Absolute Address:
Base Offset:
0x58
Size:
0x4

Description

Plaintext/Ciphertext/Stream Size (in Byte)

• The size must be no less than 8 bytes (ECB requires data to be multiple of 8 bytes).

• If the stream is divided to multiple segments (Resume Operation), all segments must be multiple of 8 bytes. If it is not ECB, the last segment allows partial codeword.
