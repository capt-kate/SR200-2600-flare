Field Descriptions

axi_data_par_en - AXI Data Bus Even Parity Check Enable

When the following bits are set, the AXI Data Bus Even Parity Check is enabled.

axi_data_par_even - AXI Data Bus Even Parity

When the following bits are set, the AXI Data Bus Even Parity is selected.

ahb_wr_seq_allow - AHB Allows Sequential Write

When this bit is set, the write address of the sequential write will not be written into FIFO. When the this bit is reset, the write address and data will be written into FIFO. This mode is reserved for AHB bus.

axi_sl_rd_prot - AXI Slave Read Protect Mode

When the following bits are set, the AXI will check the AXI Read Protect mode of the incoming Read Address Field before performing the action: AXI_SL_RD_PROT[0] = 1, only privileged access can be performed. AXI_SL_RD_PROT[1] = 1, only secured access can be performed. AXI_SL_RD_PROT[2] = 1, only data access can be performed.

axi_sl_wr_prot - AXI Slave Write Protect Mode

When the following bits are set, the AXI will check the AXI Write Protect mode of the incmming Read Address Field before performing the action: AXI_SL_WR_PROT[0] = 1, only privileged access can be performed. AXI_SL_WR_PROT[1] = 1, only secured access can be performed. AXI_SL_WR_PROT[2] = 1, only data access can be performed.

axi_sl_bypass_fifo - AXI Slave Bypass QFIFO Enable

When this bit is set, the read or write command is bypassing the AXI slave input FIFO.

axi_sl_rd_qena - AXI Slave Queue Read Operation Enable

When this bit is set, the read command is allowed to be stored in the Input FIFO interleaved w/ write command.

axi_sl_bus_sz - AXI Slave Bus Size

This register provides the maximum bus size. The value is indicated as the following: 0 - 32 bit bus 1 - 62 bit bus

out_axi_sl_word_swp - Slave AXI Output Word Swap Enable

The upper and lower word of the 64 bit data bus are swapped.

out_axi_sl_byte_swp - Slave AXI Output Byte Swap Enable

The byte locations within the 32 bit word are swapped.

in_axi_sl_word_swp - Slave AXI Input Word Swap Enable

The upper and lower word of the 64 bit data bus are swapped.

in_axi_sl_byte_swp - Slave AXI Input Byte Swap Enable

The byte locations within the 32 bit word are swapped.
