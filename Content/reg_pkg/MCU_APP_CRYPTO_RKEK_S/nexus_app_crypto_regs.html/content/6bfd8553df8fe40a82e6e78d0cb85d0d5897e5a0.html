Field Descriptions

read_pending - Input DMA Out-OF-Order Read Pending Status

This field provides the Read Data Transfer pending status. This status is used to provide when DMA can shut down w/o affecting the AXI bus protocol.

pause_cmpl - Input DMA Pause Complete

When this bit is set, the Input DMA is paused.

ll_err_code - Input DMA Link List Read Cycle Error

This field provides the Read Link List Transfer Error on the bus.

00 - No Error

01 - No EX Error

10 - Slave Error

11 - Decode Error

par_err - Input FIFO Parity Error

When this bit set, the Input DMA FIFO has data parity error.

rd_cycle_err - Read Cycle Access Error Code

This field provides the Read Data Transfer Error on the bus.

00 - No Error

01 - No EX Error

10 - Slave Error

11 - Decode Error
