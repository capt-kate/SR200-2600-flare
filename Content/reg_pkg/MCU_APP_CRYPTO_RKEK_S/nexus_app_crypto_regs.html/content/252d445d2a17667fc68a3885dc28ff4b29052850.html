Hash Status

Absolute Address:
Base Offset:
0xc
Size:
0x4

Description

This register holds status from the last hash operation.

Contents

BitsIdentifierAccessResetDecodedName
[31:4] - - - -
[3] hmac_sm_busy r 0x0 Hash HMAC SM Busy For debug. H
[2] xfer_sm_busy r 0x0 Hash Xfer SM Busy For debug. H
[1] hash_busy r 0x0 Hash Busy H
[0] hash_done rw 0x0 Hash Done H

Field Descriptions

hmac_sm_busy - Hash HMAC SM Busy For debug.

xfer_sm_busy - Hash Xfer SM Busy For debug.

hash_busy - Hash Busy

If this bit is 1, it means Hash engine is processing data in the operation.

hash_done - Hash Done

If this bit is 1, it means last hash operation finished successfully. FW needs clean this bit before issue next hash command.
